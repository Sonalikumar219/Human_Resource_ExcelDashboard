Project Overview
This project features an interactive dashboard built in Microsoft Excel designed to track and analyze human resources data. The dashboard consolidates several key performance indicators (KPIs) to provide a snapshot of organizational health and employee retention.
Key Performance Indicators (KPIs)
 * Total Employees: 1,470
 * Total Attrition: 237
 * Attrition Rate: 16%
 * Active Employees: 1,233
 * Average Age: 37
 * Job Satisfaction Rating: 2.6Dashboard Components
